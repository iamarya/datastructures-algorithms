<div _ngcontent-serverapp-c318="" class="body-text p-24"><p><span style="font-size:17pt;"><strong>Binary Tree</strong></span></p><p>&nbsp;</p><p><span style="font-size:11pt;">Let us now talk about binary trees.&nbsp;</span></p><p><span style="font-size:11pt;">Binary means two. A&nbsp;<strong>binary tree</strong> is a tree in which every node can have the&nbsp;<strong>utmost</strong>&nbsp;<strong>two</strong>&nbsp;<strong>children</strong>.</span></p><p><span style="font-size:11pt;">Thus, a binary tree can have either 0,1 or 2 children.</span></p><p><span style="font-size:11pt;">All the properties of a tree are still valid for binary trees as well. Out of the two children, one child is called the&nbsp;<strong>left child&nbsp;</strong>and one is called the<strong> right child.</strong></span></p><p>&nbsp;</p><p><span style="font-size:13.999999999999998pt;"><strong>Node of a Binary Tree</strong></span></p><p><span style="font-size:11pt;">Let us now look at the node of a binary tree. The node of a binary tree consists of three things -&nbsp;</span></p><ol><li><span style="font-size:11pt;"><strong>Data Value -&nbsp;The data it comprises.</strong></span></li><li><span style="font-size:11pt;"><strong>Left Pointer -&nbsp;Pointer to the left child. If there is no left child, the pointer is NULL.</strong></span></li><li><span style="font-size:11pt;"><strong>Right Pointer -&nbsp;Pointer to the right child. If there is no right child, the pointer is NULL.</strong></span></li></ol><p>&nbsp;</p><p style="text-align:center;"><span style="font-size:11pt;"><img src="https://files.codingninjas.in/article_images/binary-trees-dsa-new-0-1702466027.webp"></span></p><p>&nbsp;</p><p><span style="font-size:11pt;">Important points -&nbsp;</span></p><ol><li><span style="font-size:11pt;">If the right pointer and left pointer are both NULL, then the node will have no children.
